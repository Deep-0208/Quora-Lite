# Quora Lite

This is my first backend learning project.

It is a simple Quora-style post application built with:

- Node.js
- Express.js
- EJS
- method-override
- UUID

## What It Does

- Show all posts on the home page
- Create a new post
- View a single post in detail
- Edit a post
- Delete a post

## How to Run

1. Open the `Quora` folder in your terminal.
2. Install dependencies if needed:

   ```bash
   npm install
   ```

3. Start the server:

   ```bash
   node index.js
   ```

4. Open your browser and go to:

   ```
   http://localhost:8080/posts
   ```

## Project Structure

- `index.js` - main Express server
- `views/` - EJS templates
- `public/` - static files like CSS
- `schema.sql` - database schema file
